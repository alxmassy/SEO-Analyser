# SEO Analyzer Tool

A lightweight, Node.js-powered SEO Analyzer backend that allows users to evaluate the SEO performance of any public web page. It checks for critical SEO elements such as meta tags, page load time, mobile-friendliness, presence of `robots.txt` and `sitemap.xml`, and gives a normalized SEO score with actionable recommendations.

---

## 🚀 Features

- ✅ **URL Validation & Fetching**
- ⚡ **Page Load Time Measurement**
- 🏷️ **Meta Title & Description Check**
- 📱 **Basic Mobile-Friendliness Detection**
- 🤖 **robots.txt Existence Check**
- 🗺️ **sitemap.xml Existence Check**
- 📊 **SEO Scoring System with Recommendations**

---

## 🛠️ Tech Stack

- **Node.js**
- **Express.js**
- **Axios** – for HTTP requests
- **Cheerio** – for HTML parsing
- **CORS** – to enable API access from other origins
